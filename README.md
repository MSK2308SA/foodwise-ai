# 🌿 FoodWise AI 2026

A Streamlit-based food sustainability platform featuring:
- 🍽️ Food Waste Prediction with Eco-Score gamification
- 📦 Smart Pantry Expiry Manager
- 🧬 Nutrition Analyzer
- 🥗 Personalized Diet Plan Generator
- 🗺️ Community Food-Sharing P2P Map (Bengaluru)
- 📊 Sustainability Dashboard
- 🤝 NGO Donation Portal

## Run Locally

```bash
pip install -r requirements.txt
streamlit run application.py
```

## Deploy on Streamlit Cloud

1. Push this repo to GitHub
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Connect your GitHub repo
4. Set main file path to `application.py`
5. Click Deploy
